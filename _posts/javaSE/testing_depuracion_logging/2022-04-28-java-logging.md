## Logging

* Se usa para hacer depuración del código mediante ``breakpoint - Punto de Ruptura`` mediante el uso de algunas librería de ``logging`` para luego examinar esos ``logs``

* Las librerías más conocidas serían
  * Log4J
  * Logback
  * log4j2
  * Log4net
  * SLF4J
