## Herramientas de Cyberseguridad

* wpscan es una herramienta de escaneo de vulnerabilidades específica para sitios web basados en WordPress.

  Permite identificar vulnerabilidades conocidas, plugins y temas instalados, así como realizar ataques de fuerza bruta para probar la seguridad de las contraseñas.

  Para usar wpscan, puedes ejecutar el siguiente comando:

<```bash
wpscan --url <http://vulnerable-website.com> --enumerate u

```
