## AWS Conceptos

> AWS (Amazon Web Services) es una plataforma de servicios en la nube que ofrece una amplia gama de servicios para computación, almacenamiento, bases de datos, análisis, inteligencia artificial y más. Aquí hay algunos conceptos clave relacionados con AWS:

### Servicios en la Nube

#### AWS Well-Architected Framework

* Un conjunto de buenas prácticas para diseñar y operar sistemas en la nube de manera eficiente, segura y confiable.

* Contiene una herramienta que se puede utilizar para evaluar la arquitectura de una aplicación y recibir recomendaciones de mejora.

#### AWS Trusted Advisor

* Herramienta para evaluar y mejorar el entorno de AWS cuando implemente su entorno AWS y permite optimizar los recursos de AWS, mejorar la seguridad y reducir costos.
